# projeto-analise-vendas-powerbi
Análise de Vendas - Superstore

## Descrição do Projeto
Este projeto apresenta um dashboard interativo desenvolvido no Power BI para análise de desempenho de vendas de uma empresa fictícia de varejo.  O objetivo é explorar dados de vendas para identificar padrões de desempenho, lucratividade por categoria de produto e o impacto de descontos nos resultados financeiros.

## Objetivos da Análise

* Analisar a evolução das vendas ao longo do tempo
* Identificar categorias e subcategorias mais lucrativas
* Avaliar o impacto dos descontos na lucratividade
* Explorar diferenças de desempenho entre regiões

---

## Ferramentas Utilizadas

* Power BI
* DAX (Data Analysis Expressions)
* Modelagem de dados
* Visualização de dados

---

## Principais Métricas

* Receita Total
* Lucro Total
* Margem de Lucro
* Ticket Médio
* Volume de Pedidos

---

## Principais Insights

• A receita apresentou crescimento consistente ao longo dos anos analisados.

• A região West apresentou o maior volume de vendas.

• A subcategoria Copiers se destacou como a mais lucrativa.

• A subcategoria Tables apresentou prejuízo.

• Descontos mais altos tendem a reduzir a lucratividade, indicando impacto negativo na margem.
